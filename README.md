# ⚡ 电磁场粒子运动模拟器

这是一个可以“边拖边看”的电磁学小实验场。
你可以把电场、磁场和粒子放到画布里，实时观察轨迹、偏转和回旋运动。

如果你想快速体验：
- 🖱️ 直接打开 `index.html`
- 🚀 或者看更详细的启动说明：[`QUICKSTART.md`](QUICKSTART.md)

## ✨ 你可以做什么

- 🧲 拖拽添加对象：电场、磁场、粒子、装置
- ▶️ 实时播放物理过程：暂停、继续、重置
- 🎛️ 调整对象参数，马上看到运动变化
- 💾 保存、加载、导入、导出场景
- 🎯 在圆形边界附近获得相切提示和吸附（按住 `Alt/Option` 可临时关闭吸附）

## 🚀 上手只要 30 秒

1. 从左侧工具栏放置一个场和一个粒子
2. 点击播放按钮或按 `Space`
3. 拖动对象位置，观察轨迹变化
4. 右键对象（触屏用长按）打开属性面板继续微调

## ⌨️ 常用快捷键

- `Space`：播放 / 暂停
- `Delete`：删除选中对象
- `Ctrl+S`：保存场景
- `Ctrl+O`：加载场景
